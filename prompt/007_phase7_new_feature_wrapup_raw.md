# [Phase 7] 신규 피처: 물리 CSV 저장 및 가중치 기반 감성 스코어링 구현
당신은 신규 비즈니스 솔루션을 제작하는 풀스택 C++ 개발자입니다. 
아래 두 가지 기능을 구현해 주세요.

[신규 요구사항 및 명세]
1. `FileHandler::saveToCsv` 실제 물리 디스크 저장 구현:
 - 기존에 콘솔 출력(`std::cout`)만 흉내 내던 stub 함수 `saveResult()`를 완전히 제거합니다.
 - `bool saveToCsv(const vector<Feedback>& data, const string& path)` 원형을 구현하여 실제 파일 시스템에 CSV 형태로 저장되도록 하세요.
 - 파일 생성 실패 시 `false` 반환 등 에러 핸들링 코드를 철저하게 넣으세요.
 - 파일 저장 시 한글이 깨지지 않도록 처음에 UTF-8 BOM("\xEF\xBB\xBF")을 기록한 후 첫 줄 헤더(`text`)를 삽입하고 다음 줄부터 피드백 내용을 줄바꿈 처리하여 물리 파일로 출력하도록 C++ `ofstream` 기법을 적용하세요.

2. 가중치 감성 분석 (`classifyWeighted`) 적용:
 - 기존의 단순 첫 매칭 키워드 즉시 중단 및 결정 처리 로직의 문제를 극복하기 위해 가중치 점수제를 도입합니다.
 - 문장 내의 모든 긍정 키워드 수와 부정 키워드 수를 모두 카운트하여 긍정이 많으면 "긍정", 부정이 많으면 "부정", 동점 또는 발견되지 않으면 "중립"을 반환하는 똑똑한 구조의 `classifyWeighted` 함수를 `TextAnalyzer`에 심어주세요.
 - 이로 인해 가중치 분석이 혼합 텍스트에서 정확도를 획기적으로 상승시키는 결과를 나타내는지 검증하는 단위 테스트 케이스도 함께 추가해 주세요.

이 결과를 상세히 요약하여 `docs\new_feature.md` 파일로 작성해 주세요.
